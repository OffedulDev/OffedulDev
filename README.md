## 🏗️ Projects I'm proud of

### 🌐 Home Network Redesign
Took full ownership of my home network — from the fiber line down to the wireless. Replaced the ISP-provided equipment with a custom **OPNsense** router and high-end access points. The result? A network so solid that phones drop their cellular connection as soon as they join it.

*I'm working on a write-up about this for Dev.to.*

### 🖥️ Raspberry Pi 5 NAS
Built a personal NAS on a Raspberry Pi 5 running a **3-2-1 backup strategy**: two mirrored disks on the device, with everything synced nightly to a **Hetzner Storage Box**. Exposed securely to the internet via **Cloudflare Zero Trust** — browser access requires a specific email allowlist, while applications authenticate via service tokens.
